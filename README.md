## 💻 Getting Started

Want to run this project locally? Follow these steps:

### Prerequisites

Ensure you have Node.js and npm installed on your machine.
*   [Node.js](https://nodejs.org/) (Download the LTS version)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/irfantriadiginting/portfolio-website.git
    cd portfolio-website
    ```
2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

### Development Server

Run the following command to start the development server with Hot-Module-Replacement (HMR).
```bash
npm run dev
# or
yarn dev
```
Open `http://localhost:5173` (or the port specified in your terminal) to view the project in your browser.

### Building for Production

To create a production-ready build of the application:
```bash
npm run build
# or
yarn build
```
The compiled files will be generated in the `dist` directory. You can preview the production build locally using:
```bash
npm run preview
# or
yarn preview
```
